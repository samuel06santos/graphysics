# [GraPhysics](https://samuel06santos.github.io/graphysics/)
<img src="graphysics-logo-png3.png" alt="GraPhysics Logo" width="200"/>

GraPhysics é um simulador de física que utiliza computação gráfica aplicado na web.

Controles principais:
- adicionar cargas positivas e negativas;
- ligar/desligar o multímetro;
- alternar o mapa de potencial;
- mostrar/ocultar os vetores do campo;
- ativar vetores sem opacidade;
- exibir uma grade centralizada no canvas;
- acompanhar as coordenadas do mouse em relação ao centro da grade.
